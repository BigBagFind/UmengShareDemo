# UmengShareDemo
####友盟分享Demo，自定义友盟分享视图,一句代码搞定(如果对你有帮助的话，就来个Star吧)
##### 项目开发过程中的副产品，封装下，方便下次使用

##### 1.将TggShareView文件夹拖入工程中
##### 2.import "TggShareView.h"
##### 3.弹出的地方(比如button的Action里面)加入这一句代码
```
// 一句弹出不多说
    [TggShareView presentShareView];
```
